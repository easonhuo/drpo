# Figure 1 D4RL-9 external gradient diagnostic

This directory contains the plot-ready pairwise reaggregation for
`EXT-H-E7-FIG1-D4RL9-GRADIENT-COVERAGE-01`.

- `main_aggregate.csv`: equal-weight aggregate on the distance support shared by all nine dataset cells.
- `dataset_plot_data.csv`: 10-seed summaries for the 3 x 3 appendix panels.
- `MANIFEST.json`: source hashes, counts, pairwise normalization, hierarchical-bootstrap protocol, and pilot boundary.
- `sensitivity.json`: linear/log-distance interpolation and mean/median sensitivity checks.

The result remains a pilot and is not formal method-ranking, task-collapse,
support-boundary, or numerical-collapse evidence. Old Hopper-only assets are
retained unchanged under `results/FIGURE1_EXTERNAL_GRADIENT/`.
