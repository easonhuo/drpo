# E8 DRPO `(c,tau)` Scale Transfer to Qwen2.5-3B

Experiment ID: `EXT-C-E8-DRPO-CTAU-SCALE-TRANSFER-3B-01`

Status: code-first pilot implementation; not yet run.

## Question

Do the four frozen implementation-facing DRPO `(c,tau)` points selected from the
0.5B response surface remain usable on Qwen2.5-3B-Instruct when all non-scale
scientific variables are held fixed?

This protocol tests transfer robustness, not a new optimum. No point may be
added, removed, or modified after seeing 3B validation metrics.

## Frozen cells

```text
A: c=1.609437912, tau=0.125
B: c=1.897119985, tau=0.25
C: c=2.995732274, tau=0.125
D: c=4.605170186, tau=0.75
seeds: 4000, 5000
```

The exact eight cells are generated by crossing the four listed points with the
two seed offsets. This is the only Cartesian operation; the `(c,tau)` point list
itself is explicit and non-Cartesian.

## Objective and remoteness

For each negative completion,

```text
u = -mean_token_log_probability / 2
q = max(u - tau, 0)
w = exp(-c * q)
```

The policy objective is the existing E8 branch-balanced positive-minus-weighted-
negative log-probability objective. Weights are detached. The denominator is the
first-occurrence unique-negative count per prompt, never the weight sum.

## Model and data

- Model: `Qwen2.5-3B-Instruct` from pretrained base plus fresh LoRA.
- Model identity gate: local `config.json` must declare `model_type=qwen2`,
  architecture `Qwen2ForCausalLM`, and an identity field matching
  `Qwen2.5-3B-Instruct`.
- Training bank: the existing frozen model-independent E8 V2 bank.
- Evaluation: existing structurally disjoint `val.jsonl`, excluded from loss.
- Separate test split: forbidden for this pilot.

The bank being historically generated under the 0.5B experiment ID does not
make it a logged 0.5B behavior-policy dataset. This protocol uses it only under
its registered model-independent bank role.

## Optimization and horizon

The 3B base config preserves the reviewed 0.5B settings:

- LoRA, BF16, no 4-bit loading;
- micro-batch 1, gradient accumulation 8;
- learning rate `5e-5`, cosine schedule, warmup ratio `0.03`;
- maximum gradient norm `1.0`;
- fixed 1200 optimizer steps;
- Greedy/Pass@8 every 100 steps;
- Pass@64 every 200 steps;
- no early stopping.

These choices isolate model scale. A different learning rate, LoRA setting,
batch, horizon, or evaluation schedule requires a successor experiment.

## Runtime and liveness

The full run requires exactly four visible GPUs and one process slot per GPU.
Eight cells therefore run in exactly two waves. The launcher must reject fewer
or more than four selected devices and any runtime slot count other than one.

Before the full run, point A / seed 4000 runs a two-step real Qwen/PEFT/CUDA
liveness with eight validation examples. The liveness is engineering evidence
only. A PASS permits the frozen sweep; it does not count as a scientific result.

## Metrics and terminal audit

Primary paper-facing reporting is late-window Pass@8 over
`800,900,1000,1100,1200`; terminal Pass@8 is secondary. Also retain Greedy,
Pass@64, valid-expression rate, gradient/update diagnostics, weight and
surprisal diagnostics, all cell curves, logs, and failure evidence.

The terminal audit must keep distinct:

- task performance;
- valid-expression or structure diagnostics;
- NaN/Inf numerical failure;
- best checkpoint versus late-window and terminal checkpoints.

No fixed-horizon run may be called converged. Two development seeds do not
support statistical significance or formal ranking.

## Stop and closure rules

The run stops only at 1200 steps or on a recorded numerical/process failure.
Task degradation, valid-rate degradation, or apparent drift do not trigger early
stopping. Failed and partial cells must be retained and delivered.

A completed pilot requires all eight expected summaries, aggregate curve,
terminal audit, source/input hashes, resource records, and durable results-repo
delivery. Scientific closure then records whether the four-point region
transfers, partially transfers, or fails; it must not select a new 3B-specific
point without registering another experiment.
